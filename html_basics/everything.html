HTML Hyperlinks

The "a" tag is used to define links.

Here is a link for more about HTML links

target="" attribute specifies where to open the linked document.

  • _blank - opens in a new window or tab. Somebody by Bryan Adams (Opens in a new tab)
  • _self - opens in same tab as clicked in (default).
  • _parent - opens in the parent frame, where it was called. Born to Run by Bruce Springsteen (Opens in parent frame)
  • _top - opens in full body of the window.Breaks out of iframe Heartbreak Hotel by Elvis Presley
  • framename - opens in named frame
